# Kujua - Landing Page

Landing page institucional da **Kujua Tecnologia**, focada em transformacao digital com responsabilidade tecnica.

## Stack

- HTML5
- Tailwind CSS (CDN)
- JavaScript vanilla
- Bilíngue (PT-BR / EN)

## Deploy

O site esta hospedado na [Vercel](https://vercel.com) e disponivel em:

- https://kujua.vercel.app
- https://kujua.com.br (quando DNS propagado)

Para fazer deploy:

```bash
vercel --prod
```

## Estrutura

```
├── index.html          # Pagina principal (HTML + JS embutido)
├── styles.css          # Estilos customizados
├── assets/             # Logos e imagens
└── screen.png          # Screenshot do site
```
